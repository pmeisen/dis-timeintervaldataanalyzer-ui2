Download TidaIs

Download the basis package of the Time Interval Data Analysis Information System. Follow the first steps to get started and learn how to use the system, how to configure your model, and how to modify the implementation to your specific needs.

Getting Started! System Requirements, Download, Unzip, and Setup

The system needs a Java Runtime Environment (JRE) 1.6 or higher installed on the system. Fulfilling this requirement, there are only a couple of simple steps to be performed: download, unzip, and start.
